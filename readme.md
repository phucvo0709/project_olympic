<strong>Installation guide:</strong>

1: git clone https://github.com/phucvo0709/project_olympic.git

2: npm install

3: composer install

4: Copy .env.example file to .env on root folder setting config.

5: php artisan key:generate

6: php artisan migrate

7: php artisan serve
